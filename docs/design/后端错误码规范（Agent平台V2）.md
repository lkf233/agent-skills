# 后端错误码规范（Agent 平台 V2）

## 1. 目标
- 为前后端联调提供稳定错误语义。
- 错误码与接口契约一起冻结，避免同一错误多种表达。
- 所有业务异常通过 `ApiResponseObject` 返回：`code` + `message`。

## 2. 分层规则
- `0`：成功
- `1xxx`：参数与请求格式错误
- `2xxx`：认证与权限错误
- `3xxx`：资源与状态错误
- `4xxx`：外部依赖错误（模型、工具、文件存储等）
- `5xxx`：系统内部错误

## 3. 错误码明细

## 3.1 参数与请求（1xxx）
- `1001` 参数校验失败
- `1002` 请求体不能为空
- `1003` 分页参数非法
- `1004` 文件类型不支持
- `1005` 文件大小超限
- `1006` JSON 配置格式非法
- `1007` 必填字段缺失

## 3.2 认证与权限（2xxx）
- `2001` 未登录或 Token 无效
- `2002` Token 已过期
- `2003` 无权访问该资源
- `2004` 无权绑定他人资产
- `2005` 无权执行该 Agent 操作

## 3.3 资源与状态（3xxx）
- `3001` 用户不存在
- `3002` Agent 不存在
- `3003` Agent 名称重复
- `3004` Agent 状态不允许该操作
- `3005` Skill 不存在
- `3006` Tool 不存在
- `3007` 知识库不存在
- `3008` 知识库文件不存在
- `3009` 会话不存在
- `3010` 预览会话不存在
- `3011` 资源已被删除
- `3012` 资产状态不可用
- `3013` 发布校验未通过
- `3014` 会话已关闭

## 3.4 外部依赖（4xxx）
- `4001` 模型调用失败
- `4002` 模型调用超时
- `4003` 向量化失败
- `4004` 向量检索失败
- `4005` 工具调用失败
- `4006` 技能执行失败
- `4007` 文件存储失败
- `4008` 文件解析失败
- `4009` 外部服务限流

## 3.5 系统错误（5xxx）
- `5000` 系统异常
- `5001` 数据库操作失败
- `5002` 并发冲突
- `5003` 数据一致性校验失败
- `5004` 消息写入失败
- `5005` 日志写入失败

## 4. 返回约定
- `message` 面向用户与前端可读，不返回堆栈信息。
- 日志中记录内部异常原因，外部响应只返回约定错误码与简要文案。
- 同一错误场景必须返回固定错误码，不允许随机复用。

## 5. 接口映射建议
- 认证登录失败：`2001`/`2002`
- Agent 发布前校验失败：`3013`
- 预览会话调用关闭会话：`3014`
- 工具测试失败：`4005`
- 向量检索失败：`4004`

## 6. 治理规则
- 新增错误码时，必须更新本文件并同步 OpenAPI 文档说明。
- 禁止复用已有错误码表达新语义。
- 发现历史错误码语义冲突时，新增新码并标记旧码废弃周期。
